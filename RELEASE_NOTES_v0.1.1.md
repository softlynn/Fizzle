# Fizzle v0.1.1

This update focuses on UI cohesion, update controls, and virtual mic clarity.

## Highlights
- New cohesive top window header with integrated Fizzle logo.
- Salmon-toned minimize/maximize/close controls while keeping the same behavior.
- Settings now has dedicated tabs:
  - App Auto-Enable
  - Updates
- Updates tab includes:
  - Auto-download toggle
  - Check updates button
  - One-click links to GitHub repo and website
- About dialog now includes quick links to GitHub and website.
- Faster settings panel fade animation.
- Added subtle click-fade accents on Save Preset and About.
- Added subtle version watermark in the lower-right corner.
- Virtual mic UI now uses Fizzle branding:
  - Output label uses Fizzle Mic naming
  - Status text uses Fizzle Mic naming
  - Virtual mic icon uses Fizzle logo
