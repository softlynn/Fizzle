# Fizzle v0.1.8

## Highlights
- Improved launch/reconfigure stability for audio devices and buffer size changes.
- Appearance settings now apply live: Light/Dark, Aqua/Salmon, Transparent/Opaque background, and UI size scaling.
- Smoother window dragging and header polish with cleaner divider motion.
- Tray popup rendering refinements to reduce border artifacts.
- Runtime diagnostics now report actual active callback buffer size.
