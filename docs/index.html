Windows Mic FX

Fizzle

Beautifully simple microphone processing with tray controls, presets, and VST3 plugins.

Download Latest GitHub

Install in 3 Steps

  1. 1Download the latest installer.
  2. 2Run Fizzle-Setup.
  3. 3Open Fizzle and follow the first-launch guide.

First Launch

Pick your input mic, choose your virtual output, turn Effects On, then add your VST3 plugins.

Everyday Use

Open from tray, switch presets fast, and use Restart Audio when your device setup changes.

Why People Like It

Low-latency path, clean UI, smooth controls, and a workflow focused on fast voice setup.

Troubleshooting

If audio crackles, increase buffer size. If apps hear the wrong mic, re-check input/output selection.
