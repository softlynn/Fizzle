Daily Workflow

Open from tray, switch presets quickly, and use Restart Audio whenever your device list changes.

Startup Controls

Use Settings → Startup to launch with Windows, start in tray, and follow app auto-enable state.

Smooth Control

Reorder plugins, adjust wet/dry, and keep a clean low-noise signal path.

Monitoring

Use Listen for local monitoring and quickly validate your chain before jumping into calls.

Troubleshooting

If you hear crackles, raise buffer size. If apps don’t hear Fizzle, verify mic input selection in-app.

Quick FAQ

Where do I get updates?

Use the Download button above, or open the latest release page on GitHub.

Can I keep Fizzle running in the tray?

Yes. Fizzle is designed for tray-first use so you can stay in your app/game.

Can Fizzle start with Windows?

Yes. Open Settings → Startup to enable Windows startup and tray launch behavior.

Do I need to configure everything every time?

No. Save presets once and switch instantly when needed.

Can Fizzle update automatically?

Yes. Open Settings → Updates and enable Auto-install new updates to download and apply updates automatically.
