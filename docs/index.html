Fizzle

A clean, tray-friendly microphone effects app for Windows.

Download Latest View on GitHub

Install

  1. Download the latest installer.
  2. Run Fizzle-Setup.
  3. Open Fizzle from Start Menu or Desktop.

First Launch

  1. Select your microphone in Input Mic.
  2. Select your virtual mic in Virtual Mic Output.
  3. Turn Effects On.
  4. Add your VST3 plugins and save a preset.

What You Get

Realtime mic processing, tray controls, easy preset switching, drag-and-drop VST order, and quick monitoring with Listen.

Need Help?

If audio sounds wrong, increase buffer size and verify your app is using the same virtual mic output selected in Fizzle.
