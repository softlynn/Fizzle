## Fizzle v0.1.0

First public release of Fizzle, now with a polished cohesive UI update.

### Features
- Tray-first microphone processing app for Windows.
- Real-time VST3 plugin chain with drag reorder and wet/dry control.
- Preset save/load/delete workflow.
- Virtual mic routing and listen monitoring.
- Diagnostics panel (latency, CPU, levels, dropped buffers).
- First-launch in-app quick setup guide.
- Installer with Start Menu and desktop shortcut support.

### UI & Experience Update
- Refined liquid-glass styling across the app.
- Smoother control rendering (buttons, dropdowns, sliders, knobs).
- Cohesive blue brand palette and improved visual hierarchy.
- Cleaner responsive website layout with improved motion and readability.
