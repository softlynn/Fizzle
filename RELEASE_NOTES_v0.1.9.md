# Fizzle v0.1.9

## Highlights
- Fixed transparent mode fallback behavior to prevent white background failures on some Windows setups.
- Added settings panel scrolling for small window sizes.
- Removed duplicate external resize affordance and kept a single in-app resize grip.
- Reduced idle GPU usage with adaptive UI timer throttling.
- Improved light-mode hover/click visibility.
- Added GitHub Pages favicon support.
